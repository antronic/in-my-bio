# in-my-bio

Personal links page built with Astro.

## Project Structure

```
/
├── src/
│   ├── components/
│   │   └── Card.astro
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
├── public/
└── package.json
```

## Commands

| Command     | Action                           |
| :---------- | :------------------------------- |
| `bun dev`   | Start dev server at localhost:4321 |
| `bun build` | Build production site to ./dist  |
| `bun preview` | Preview build locally           |

## Stack

- Astro 5.x
- TypeScript
- Bun

## Development

```bash
bun install
bun dev
```
